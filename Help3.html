Formulaire

TimesUpdate : mettre à jour le fichier netCDF de temps
(Re)Créer le fichier *_times.nc TimesUpdate -r *_times.nc fichier1.nc fichier2.nc ... fichierN.nc
Mettre à jour le fichier *_times.nc TimesUpdate -u *_times.nc fichier1.nc fichier2.nc ... fichierN.nc
Mettre à 0 la ligne représentant un fichier référencé dans *_times.nc TimesUpdate -d *_times.nc mauvais_fichier.nc
Supprimer définitivement un fichier référencé dans *_times.nc DDClearTimes *_times.nc


nc2nc : corriger start/stop times
Corriger start/stop times dans un fichier netCDF de données nc2nc fichier.nc -bStartTime -eStopTime -gzip/unzip


infoLocal2nc : créer le fichier netCDF d'informations
Créer le fichier netCDF d'informations à partir du fichier XML homologue infoLocal2nc *_info.xml *_info.nc


UpdateInfo : mettre à jour le fichier netCDF d'informations
Mettre à jour Start/Stop time sur MANUNJA php /home/budnik/AMDANEW/DDLIB/bin/UpdateInfo.php prefixe_fichiers_info
Mettre à jour Start/Stop time sur CDPP1 php /usr/local/AMDA/DDLIB/bin/UpdateInfo.php prefixe_fichiers_info


makeDDsys : créer/mettre à jour refer.nc
Créer/mettre à jour le fichier netCDF qui rescence tous les datasets : refer.nc makeDDsys


AmdaUpdate : mettre à jour l'arbre AMDA

Faire une mise à jour de l'arbre AMDA sur APUS en installant un nouveau jeu de données sudo php /home/natacha/AMDA-NG/trunk/php/AmdaUpdate.php install

SPICE kernels
SPK (Speed/Position Kernel) Vitesse et position des objets célestes
PCK (Planetary Constant Kernel) Taille, forme, orientation des palnètes du système solaire
CK (C-Matrix Kernel) Orientation du vaisseau spatial
FK (Frame Kernel) Placement/changement de référentiel
